# 요청 메소드

- GET 			: doGet
* 용도			: 조회
* 데이터 			: URL 파라미터로 전달 
* 본문(body)		: X(사용 안함)
* 특징			: URL 길이제한, 민감한 정보 전달 금지(정보노출)

- POST 			: doPost
* 용도			: 등록
* 데이터 			: 본문에 지정 
* 본문(body)		: O 
* 특징			: form 요청(등록 요청), 로그인 

- PUT			: doPut
* 용도			: 수정
* 데이터 			: 본문에 지정 
* 본문(body)		: O 
* 특징			: 비동기 요청(수정 요청)

- DELETE		: doDelete
* 용도			: 삭제
* 데이터 			: URL 파라미터로 전달 
* 본문(body)		: X, (일부 허용)
* 특징			: 삭제 요청 전 확인 절차 필요
